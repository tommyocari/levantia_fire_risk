# WildFireWatch 🔥

A climate risk assessment pipeline and interactive dashboard for wildfire prediction in the fictional region of **Levantia**.

This project simulates the work of a climate data scientist at a risk analytics firm. Starting from synthetic climate and fire event data, it builds a full end-to-end pipeline — from data generation and feature engineering to machine learning models and an interactive dashboard.

## What this project covers

- Generating and validating large-scale scientific datasets (NetCDF, GeoTIFF)
- Building a feature engineering pipeline with xarray and pandas
- Training wildfire occurrence and burned area prediction models
- Producing geospatial risk maps and animations
- Deploying an interactive dashboard with real-time what-if scenario analysis

## Fictional setting

You are a data scientist hired by the regional government of Levantia, a Mediterranean-like territory prone to summer wildfires. Your mission: turn 10 years of climate and fire event data into an actionable risk product that officials can use without writing a single line of code.
